🛒 Amazon Application – Manual Testing Project

📘 Project Overview

This project focuses on manual testing of an Amazon-like eCommerce web application.

The primary objective is to validate that the core user journey works seamlessly, ensuring a smooth and reliable experience for end users.

The testing covers key functionalities such as:

* User Login
  
* Product Search
  
* Add to Cart
  
* Apply Promo Code
  
* Place Order

All test cases are positive scenarios, designed to verify that the system behaves correctly under valid conditions.

⚙️ Test Scenarios

 1️⃣ Login Functionality

Precondition: User is logged out

Steps:

* Open the application
  
* Enter valid username and password

Expected Result:

User should be successfully logged in and redirected to the home page



 2️⃣ Search Product
 
Precondition: User is logged in

Steps:

* Enter product name in search bar

Expected Result:

Relevant search results should be displayed



3️⃣ Add Product to Cart

Precondition: User is logged in and product page is open

Steps:

* Navigate to product page
  
* Click "Add to Cart"

Expected Result:

Product should be added to cart successfully



4️⃣ Apply Promo Code

Precondition:User is in checkout process

Steps:

* Enter valid promo code (e.g., "WINTER20")

Expected Result:

Discount should be applied successfully


 5️⃣ Place Order

Precondition: Product is added to cart

Steps:

* Click "Place Order"

Expected Result:

Order should be placed and confirmation message displayed
